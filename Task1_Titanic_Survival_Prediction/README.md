# 🚢 Titanic Survival Prediction

## 📌 Project Description

This project was developed as part of the CODEVEDX Data Science Internship.

The objective of this project is to predict whether a passenger survived the Titanic disaster using Machine Learning based on passenger details.

---

## 🎯 Objective

To build a classification model that predicts passenger survival.

---

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn

---

## 🤖 Machine Learning Algorithm

* Random Forest Classifier

---

## 📊 Dataset Features

* Passenger Class
* Gender
* Age
* Siblings/Spouses
* Parents/Children
* Fare
* Embarked

---

## 🚀 Project Workflow

1. Load Dataset
2. Handle Missing Values
3. Encode Categorical Data
4. Train-Test Split
5. Train Random Forest Model
6. Predict Survival
7. Evaluate Model Performance

---

## 📈 Results

* Accuracy: 82.68%
* Successfully predicted passenger survival.
