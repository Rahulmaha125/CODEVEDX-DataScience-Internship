# 🌸 Iris Flower Classification

## 📌 Project Description

This project is developed as part of the CODEVEDX Data Science Internship.

The objective of this project is to classify Iris flowers into three species using Machine Learning techniques based on their sepal and petal measurements.

## 🎯 Objective

To build a machine learning model that accurately predicts the species of an Iris flower.

### Species Classification

* Iris Setosa
* Iris Versicolor
* Iris Virginica

---

## 🛠 Technologies Used

* Python
* NumPy
* Pandas
* Scikit-Learn

---

## 🤖 Machine Learning Algorithm

Random Forest Classifier

---

## 📊 Dataset Details

The Iris dataset contains:

* 150 Samples
* 4 Features
* 3 Target Classes

### Features

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

---

## 🚀 Project Workflow

1. Load Iris Dataset
2. Data Preprocessing
3. Train-Test Split
4. Model Training
5. Prediction
6. Model Evaluation

---

## 📈 Results

* Accuracy Score: 100%
* Successful Species Classification

### Sample Prediction

Input:

[5.1, 3.5, 1.4, 0.2]

Output:

Setosa
